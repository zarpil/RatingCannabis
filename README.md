# TrichAi 🌱

Un analizador inteligente de derivados del cannabis impulsado por inteligencia artificial.

💖 **Creado con amor por zarpil para yly** 💖

## 🚀 Guía de Instalación

1. **Clona o descarga el repositorio** en tu ordenador.
2. **Instala las dependencias** abriendo la terminal en la carpeta del proyecto y ejecutando:
   ```bash
   npm install
   ```
3. **Configura la API Key de Google Gemini:**
   - Abre el archivo `.env` que se encuentra en la raíz del proyecto.
   - Reemplaza el texto con tu clave real (puedes conseguirla gratis en [Google AI Studio](https://aistudio.google.com/app/apikey)):
     ```env
     VITE_GEMINI_API_KEY=TuClaveRealAquí
     ```
4. **Arranca el proyecto:**
   ```bash
   npm run dev
   ```
5. ¡Listo! Abre `http://localhost:5173` en tu navegador.
